🔍 Debug Steps:

  1. Click one of the test links above
  2. Open Browser Developer Tools (F12)
  3. Go to Console tab
  4. Look for these debug messages:
    • "Zego Token Generation Parameters"
    • "Generated Token: Success/Failed"
    • Any error messages
  5. Try to join the consultation
  6. Check console for any errors

❌ Common Issues:

  • Missing URL parameters: Make sure URL has room_id, username, userid
  • Invalid credentials: App ID or Server Secret might be wrong
  • Network issues: Check internet connection
  • Browser permissions: Allow camera and microphone access

  1. Page loads with Kauvery Hospital branding
