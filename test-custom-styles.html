❓ Troubleshooting Styles

If styles are NOT working:

  • Check Console - Look for style application messages
  • Inspect Elements - Right-click on Zego interface → "Inspect Element"
  • Check CSS - Look for styles with "kauvery-zego-styles" ID in <head>
  • Refresh page - Sometimes styles need a refresh to apply

If styles ARE working:

  • ✅ Background should be purple gradient
  • ✅ Video borders should be pink (#EE2D67)
  • ✅ Buttons should turn pink on hover
  • ✅ Branding should show "Kauvery Hospital" overlay

🔧 Developer Notes

Custom styles are applied using:

  • CSS attribute selectors - Target Zego elements by class patterns
  • !important declarations - Override Zego's default styles
  • MutationObserver - Reapply styles when Zego adds new elements
  • Delayed application - Styles applied after Zego loads

Style files:

  • VideoConsultation.js - Contains the applyKauveryStyles() function
  • Styles are injected as <style id="kauvery-zego-styles">
